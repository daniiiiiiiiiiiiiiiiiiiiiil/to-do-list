# Менеджер задач на Tkinter
Это простой графический менеджер задач (To-Do List), разработанный на Python с использованием библиотеки Tkinter. Проект предназначен для лабораторной работы по работе с GitHub в группе из трёх человек. Он позволяет добавлять задачи, удалять их и сохранять список в файл JSON.

## Описание
Приложение состоит из трёх модулей:

- **gui.py**: Отвечает за создание графического интерфейса (окно, поле ввода, кнопки, список задач).

- **logic.py**: Содержит функции для добавления, удаления задач и обновления списка.

- **main.py**: Собирает всё вместе, обрабатывает загрузку/сохранение задач из файла и запускает приложение.

## Функционал:
- Добавление новой задачи через поле ввода.

- Удаление выбранной задачи из списка.

- Автоматическое сохранение задач в **tasks.json** при закрытии окна.

- Загрузка задач при запуске.

## Требования
- Python 3.x (Tkinter входит в стандартную библиотеку Python).

- Нет дополнительных зависимостей (кроме стандартных: **tkinter, json, messagebox**).

## Установка
1. Клонируйте репозиторий:
```
git clone https://github.com/https://github.com/daniiiiiiiiiiiiiiiiiiiiiil/to-do-list.git
```
2. Перейдите в директорию проекта:
```
cd to-do-list
```
## Использование
Запустите приложение с помощью:
```python
python main.py
```

- Введите задачу в поле и нажмите "Добавить задачу".

- Выберите задачу в списке и нажмите "Удалить задачу".

- Задачи сохраняются в **tasks.json** автоматически.
